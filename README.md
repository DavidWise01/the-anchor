# the-anchor

THE ANCHOR — external anchoring, the ACI CORE property that separates a witnessed claim from a generated one. Two claims: one tethered to a witness (sha256(claim|witness), re-verified live), one floating. Tamper it or forge an anchor and the recomputation catches it. The binding is real; the externality is the flagged figure.

**Live:** https://davidwise01.github.io/the-anchor/
